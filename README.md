BukkitPE is a server software for Minecraft: Pocket Edition, Which allows you to create a Minecraft Pocket Edtion server. BukkitPE aim for Clean feels and Stable Features / Performance optimization. 

# FEATURES
- [x] Performance optimization (Our #1 Goal!)
- [x] XP
- [x] Weather
- [x] Mobs AI (Including Animals)
- [x] Potions
- [x] Fast Chunk loading and sending
- [x] Redstone
- [x] Effects
- [x] Nether
- [x] Plugin API (So you can extend it)
